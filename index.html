Lokaler Generator

Apple-Konfigurationsprofil für Web-Sperrlisten

Diese WebApp erzeugt im Browser ein standardnahes .mobileconfig-Profil mit com.apple.webcontent-filter-Payload. Alle Eingaben bleiben lokal auf diesem Gerät.

Keine Server-Kommunikation Download direkt im Browser Primär für iPhone / iOS gedacht

Hinweise

Transparenz und Installation

Lokale Verarbeitung

Alle Verarbeitungsschritte laufen ausschließlich in Ihrem Browser. Es werden keine Eingaben an einen Server oder an eine externe API gesendet.

Apple-Kontext

Die App erzeugt einen standardnahen Generator für ein Apple-Konfigurationsprofil mit Web-Content-Filter-Payload. Ob die Sperren auf einem iPhone tatsächlich durchgesetzt werden, hängt vom unterstützten Verwaltungs- und Enrollment-Kontext des Zielgeräts ab. Die referenzierte Payload-Dokumentation beschreibt den Built-In-Filter insbesondere für überwachte Geräte.

Installation

Profil herunterladen, auf dem iPhone öffnen und die Installation prüfen. Anschließend unter Einstellungen > Allgemein > VPN & Geräteverwaltung nachsehen, ob das Profil erkannt wurde. Eine erfolgreiche Installation bedeutet nicht automatisch, dass die Sperrwirkung in jedem privaten Szenario erzwungen wird.

Entfernung

Nach dem Entfernen des Profils sollen die gesetzten Einschränkungen wieder entfallen. Das ist als praktischer Hinweis zu verstehen, nicht als Garantie für jede iPhone- oder Verwaltungsumgebung.
